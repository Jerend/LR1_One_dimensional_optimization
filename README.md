# LR1-One-dimensional-optimization

Установка зависимостей
```bash
pip install -r requirements.txt
```

Запуск
```bash
python main.py
```

Использование
```bash
Выбирается один из методов (1-3) или изменение параметра (4-6)
После выбора метода нужно выбрать функцию (1-3)
После выбора изменения параметра нужно ввести новое значение для параметра
```
