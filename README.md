# LR1-One-dimensional-optimization

Установка зависимостей
```bash
pip install -r requirements.txt
```

Запуск
```bash
python main.py
```

Использование
```bash
1 - Дихотомический поиск

2 - Золотое сечение

3 - Метод Фибоначчи

4 - Изменить параметр e

5 - Изменить параметр l

6 - Изменить интервал [a; b]

7 - Выход
```

При вводе параметров:
```bash
Enter - подтвердить ввод

Esc - отмена
```

На графике:
```bash
Зеленая вертикальная область - начальный интервал неопределенности

Красная вертикальная область - конечный интервал неопределенности

Зеленая точка - оптимальное значение на начальном интервале

Красная точка - оптимальное значение на конечном интервале

Синяя линия - график выбранной функции
```
